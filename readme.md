# mdpd

embed pd patches in markdown

# usage

```bash
yarn dev <md file>
```

# inputs

- hsl
- bang
- floatatom

# installation

1. install node v18 or higher
2. install webpd cli
3. clone this repo
4. install dependancies (using npm or yarn)

- hsl
- bng
- floatatom
